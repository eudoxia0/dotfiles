# Start Function
#
# The start function is run right after fvwm is done reading
# the config file. This function run after each restart
# so using Test (Init) or Test (Restart) can further control
# actions that are run during the first time run (Init) or
# actions that are run after a restart.
DestroyFunc StartFunction
AddToFunc   StartFunction
+ I Test (Init) InitBackground
+ I Module FvwmEvent EventNewDesk
+ I Module FvwmButtons MainPanel

# Function to set background when fvwm starts
DestroyFunc InitBackground
AddToFunc InitBackground
+ I Exec exec feh --bg-fill $[infostore.walls]/osx-panther.png

# Mouse Bindings Functions

+ M $0
+ D $1

DestroyFunc RaiseMove

+ M $0

#
# Mouse Bindings
#

# super + left click hold: move window
Mouse 1 W 4 Move

# super + right click hold: resize window
Mouse 3 W 4 Resize

# left click on the rightmost window button: close
Mouse 1 2 A Close

# Click on the titlebar to Raise, Move, Double Click to Maximize.
Mouse 1 T    A RaiseMoveX Move Maximize

# Mouse Wheel Up/Down on titlebar to WindowShade On/Off
Mouse 4 T    A WindowShade True
Mouse 5 T    A WindowShade False

# Borders: Click to raise, Move to Resize
Mouse 1 FS   A RaiseMove Resize

# Left click on the root window: nothing.
Mouse 1 R    A Nop

# Right click on the root window: launch the FVWM menu.
Mouse 3 R    A Menu MenuFvwmRoot

# Middle click on the root window: open the window list.
Mouse 2 R    A WindowList
