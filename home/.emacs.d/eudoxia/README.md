# Emacs Cheat Sheet

System:

| Key       | Action |
|-----------|--------|
| `C-x C-c` | Quit   |

Files:

| Key       | Action    |
|-----------|-----------|
| `C-x C-f` | Find file |
| `C-x C-s` | Save      |

Movement:

| Key   | Action          |
|-------|-----------------|
| `C-b` | Backward char   |
| `C-f` | Forward char    |
| `M-b` | Backward word   |
| `M-f` | Forward word    |
| `C-a` | Start of line   |
| `C-e` | End of line     |
| `C-p` | Previous line   |
| `C-n` | Next line       |
| `M-<` | Start of buffer |
| `M->` | End of buffer   |

Clipboard:

| Key   | Action |
|-------|--------|
| `C-w` | Cut    |
| `M-w` | Copy   |
| `C-y` | Paste  |

Delete:

| Key             | Action              |
|-----------------|---------------------|
| `C-d`           | Kill next char      |
| `M-d`           | Kill next word      |
| `M-<backspace>` | Kill previous word  |
| `C-k`           | Kill to end of line |

Undo:

| Key   | Action |
|-------|--------|
| `C-/` | Undo   |

Treemacs:

| Key         | Action              |
| `F8`        | Toggle Treemacs     |
| `C-c C-w e` | Edit workspace list |
